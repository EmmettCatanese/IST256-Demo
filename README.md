# JupyterLite Demo

## This is my demo of my IST 256 Project!

IST 256 is titled Introduction to Introduction to Python for the Information Profession. Most of it is learning the basics of Python and then at the end we learn some data analysis libraries and tools using python.

## For our final project we were tasked with telling a story with data
I struggled to decide what dataset I wanted to use but I have been watching a lot of baseball lately so I landed on using a baseball dataset. The dataset I landed on using (Retrosheet) stored each game as a row and then a bunch of different attributes based on it. From this, I did a variety of analysis and then made this final project with a interact widget

## How can I use this repository?
This repo is a fork of Juypter Lite which runs Juypter notebook right in your browser! If you would like to use this app just head over to https://emmettcatanese.com/IST256-Demo/lab/index.html and then run the code! Below the code cell you will get the interact widget. If you would like to look over the code and the other datasets, you can do that right in Juypter Light or head over to the content folder in this repo where you can download the files.
